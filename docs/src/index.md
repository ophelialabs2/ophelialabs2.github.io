# MyDocumentedPackage.jl

A documentation template built with Documenter.jl.

## Features

- Simple and elegant documentation structure
- Automated deployment with GitHub Actions
- Professional HTML output

## Quick Start

```julia
using MyDocumentedPackage

hello("World")
add(2, 3)
```

## Installation

```julia
using Pkg
Pkg.add("https://github.com/ophelialabs2/ophelialabs2.github.io")
```

See the [Getting Started Guide](manual/getting_started.md) for more detailed information.
